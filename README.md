### 🔍 Vision-to-JSON API
> *Python Capstone Project — Master's Program*

A high-performance **FastAPI** service that uses **GPT-4o Vision** to analyze complex documents (invoices, forms, contracts) and returns structured, validated JSON data. Built as a real production-ready microservice.

**Stack:** `Python` `FastAPI` `OpenAI API` `Pydantic` `REST`

**What it does:**
- Accepts image or PDF input via API endpoint
- Sends to GPT-4o Vision for document understanding
- Validates and returns clean, typed JSON responses
- Handles edge cases with Pydantic schema enforcement
